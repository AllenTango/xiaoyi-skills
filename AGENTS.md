# AGENTS - xiaoyi-skills

进入仓库后先读取根 `STATUS.md`，再读取目标技能目录内的 `SKILL.md`、`AGENTS.md` 和 `STATUS.md`。

## 工作约定

1. 技能修改遵守 skill-creator 规范，保持 `SKILL.md` 简洁并使用渐进披露。
2. 不覆盖用户已有的未提交变化；修改前后检查 Git 状态和 diff。
3. 修改扫描器或生成脚本时补充自动化测试并真实运行。
4. 完成实质性工作后同步根项目与目标子技能的状态事实。
5. 未验证的技能行为不得标记为完成。

## 验证

- 对目标技能运行其测试命令。
- 使用 `quick_validate.py` 验证技能结构。
- 使用实际任务进行至少一次行为验收。
